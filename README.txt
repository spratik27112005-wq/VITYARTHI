This repository contains 34 fully commented Python programs, each of which is designed to explore mathematical and number theory concepts, algorithmic approaches, and computational challenges. All scripts are written in Python 3 and are best used in VS Code or any modern Python IDE. 

Program Categories and Descriptions
Primality & Special Primes

Fibonacci-Prime Check: This checks whether a number is both a Fibonacci and a prime number.

Twin Prime Finder: Finds primes whose difference is exactly two.

Mersenne Prime Tester: Tests numbers of the form 2^n-1 for primality.

Prime Power Validator: This identifies numbers that are powers of primes.

DISTINCT PRIME FACTORIZER Identifies all different prime factors of a number.

Pollard Rho Prime Factorization: Uses Pollard’s Rho algorithm to factor large numbers.

Miller-Rabin Primality Test: A probabilistic primality test for large integers.

Arithmetic & Modular Functions

Chinese Remainder Theorem Solver. Solves systems of modular congruences.

Modular Multiplicative Inverse: Finds modular inverses using Extended Euclidean Algorithm.

Modulo n Order Calculator: Calculates the least k such that a^k ≡ 1 mod n.

Euler's Totient Function: Counts numbers coprime to n.

Möbius Function Evaluator: Evaluates μ(n) given input n.

Quadratic Residue Checker: Checks whether a is a quadratic residue mod p.

Carmichael Number Detection: Tests whether a number is composite with Carmichael property.

Combinatorial & Partitioning

Partition Function Calculator: Counts ways of expressing n as sums of positive integers.

Divisor Counter: Counts the number of divisors of a given number.

Collatz Sequence Length: This program calculates the number of steps required to arrive at 1 in the Collatz sequence.

Perfect Powers Checker: Checks whether a number can be written as a^b for a, b > 1.

Sequences & Structure

Lucas Numbers Generator: Generates first n terms of a Lucas sequence.

Polygonal Numbers Calculator: This program calculates the nth s-gon numbers for user inputs.

Factorial Calculator: Calculates n! using iteration.

Automorphic Number Checker: Finds numbers whose square ends with the number itself.

Harshad Number Checker: Checks for numbers that are divisible by the sum of their digits.

Deficient Number Checker: Checks whether divisor sum is less than 2n.

Amicable Number Checker: Finds pairs of numbers whose divisor sums match each other.

Multiplicative Persistence: Finds the steps to reach one digit via digit multiplication.

Other & Miscellaneous

Perfect Power Classification: This program classifies numbers into perfect powers.

Chronic Number Checker: Evaluates specific number property.

Approximate Pi Calculator: Estimates the value of Pi using a numeric method.

Lagrange Interpolation: Implements Lagrange's method for numeric interpolation.

General Zeta Function Approximator Approximates values of zeta function.

Utility: Count Divisors counts divisors using both a brute force approach and an optimized one.

Usage All programs run with Python 3 on VS Code. Each of them has main test cases and can be edited for custom input. Results will print to the integrated terminal. For algorithms requiring visualization or specific inputs, refer to comments in the code for their usage.
